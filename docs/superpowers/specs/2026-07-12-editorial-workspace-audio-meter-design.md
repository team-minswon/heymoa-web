# Editorial Workspace and Audio Meter Design

## Goal

워크스페이스 홈을 조밀한 편집형 회의록 목록으로 정돈하고, 실제 마이크 음성에 즉각 반응하는 파형을 제공한다.

## Workspace home

- 큰 흰색 목록 카드를 제거하고 날짜 구분선과 노트 행을 캔버스 위에 직접 배치한다.
- 콘텐츠는 최대 너비를 넓히고 제목, 날짜, 노트 제목의 왼쪽 기준선을 일치시킨다.
- 각 행은 `파형·재생 시간 / 제목·폴더 / 기록 시각·작성자` 순으로 정보를 제공한다.
- 행은 기본적으로 평평하게 두고 hover, focus, 선택 상태에서만 옅은 배경과 둥근 모서리를 드러낸다.
- 모바일에서는 작성자 메타데이터를 숨기고 핵심 제목, 폴더, 시간만 유지한다.
- 현재 API에 없는 캘린더 연동, 공유, 자동 문서 기능은 추가하지 않는다.

## Side note surface

- 기존 `view=side` URL과 겹치는 패널 방식을 유지한다.
- 패널의 둥근 모서리, hairline, warm-white 표면은 왼쪽 사이드바와 같은 시각 언어를 사용한다.
- 배경 목록과 패널이 동시에 읽히도록 데스크톱에서 약 절반의 폭을 차지한다.

## Microphone level

- Web Audio `AnalyserNode`의 RMS 원본값에 noise floor와 음성 범위 gain을 적용해 지각 가능한 `0..1` 값으로 정규화한다.
- 빠르게 올라가고 천천히 내려가는 attack/release smoothing으로 떨림을 줄인다.
- 최근 레벨 이력을 작은 고정 길이 배열로 유지해 정적인 막대 배율 대신 실제 입력의 시간 흐름을 표현한다.
- 녹음 중인 노트 행, 전역 표시기, 전사 화면은 동일한 정규화 레벨을 사용한다.
- 일시정지와 종료 시 레벨 및 이력을 0으로 되돌린다.

## Accessibility and tests

- 파형은 `role="meter"`, 한국어 label, `aria-valuenow`를 유지한다.
- 레벨 정규화는 무음, noise floor, 일반 음성, 큰 입력을 단위 테스트한다.
- 워크스페이스 목록은 날짜 그룹과 노트 링크가 카드 wrapper 없이 렌더링되는지 검증한다.
- 기존 OpenAPI, AsyncAPI, Orval 생성물은 변경하지 않는다.
