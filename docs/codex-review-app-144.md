# codex 리뷰 기록 — APP-144

게이트는 로컬 `codex exec review --base dev` 하나. 문서 하나짜리 변경이라 merge 직전 한 번.

| 지적 | 판단 | 근거 |
|---|---|---|
| **P2** 분석 필드를 "파싱하지 않고 그대로 렌더" | 고침 | 문장이 모호했다. **서버가** 구조화하지 않는다는 뜻이었는데 **web이** 원문을 그대로 출력하라는 말로 읽힌다 — 그러면 `#`·`-`가 사용자에게 노출되고, 같은 문서의 `Tabs + markdown 렌더` 매핑과도 어긋난다 |
| **P2** 승인 API의 409 `MEETING_NOT_ACTIVE` 누락 | 고침 | 계약을 다시 보니 정확했다. 승인을 누르는 사이 회의가 끝난 경우다. 일반 오류로 뭉개면 카드가 계속 눌리는 상태로 남는다 |

둘 다 **문서가 만들 오해**를 잡은 지적이다. 이 문서는 여섯 에이전트와 일곱 화면 이슈가
읽는 공통 입력이라, 모호한 문장 하나가 여러 곳에서 같은 방식으로 어긋난다.
